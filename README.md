# Braintly Technical Challenge

Welcome, and thanks for taking the time to do this challenge!

We're excited to have you here. This is an opportunity for you to show how you
work — how you reason about problems, structure a solution, and communicate your
decisions. We value clarity and honest trade-offs over a "perfect" answer.

Inside this repository you'll find a folder for each exercise we've asked you to
solve. Each folder has its own `README.md` with the full instructions, and (where
relevant) a `SETUP.md` explaining how to run it.

We wish you the best of luck and happy coding! 🙌
